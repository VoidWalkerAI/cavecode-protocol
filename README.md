CaveCode is a plain-text blueprint format for describing systems, tools, utilities, and games in a way that:

humans can read and edit on a phone

AIs can consistently generate and parse

teams can remix, validate, and build upon

multiple engines (OpenAI, Claude, Gemini, etc.) can speak a shared standard


It is not a programming language.
It sits beside the code as a structured description: numbered blocks with clear labels, emoji glyphs, and a dedicated human edit zone.
